## Environment:
- Java version: 1.8
- Maven version: 3.*
- Spring Boot version: 2.2.1.RELEASE

## How to test:
```bash
curl http://localhost:8080/v1/invitations
```

## Commands
- run: 
```bash
mvn clean package; java -jar target/hubspot-assessment-1.0.jar
```
- install: 
```bash
mvn clean install
```
- test: 
```bash
mvn clean test
```
